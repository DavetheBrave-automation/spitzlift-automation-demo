06 🔐 Security Disclosure — Findings 1 HIGH · 1 MED
HIGH · CVSS 8.8 Unauthenticated File Manager API Endpoint Exposed

Plugin: File Manager Advanced — REST namespace file-manager-advanced/v1 visible at spitzlift.com/wp-json/ without authentication. Discovered via passive enumeration only — no endpoints called, no data accessed.

  • Unauthenticated attacker may browse server file/directory structure
  • wp-config.php potentially readable → database hostname, name, username, password in plain text
  • If DB creds obtained: full read/write on all WooCommerce data, all form submissions, all distributor/customer records
  • Worst case: unauthenticated file upload → remote code execution on server
  • Automated scanners (WPScan, Shodan) probe millions of WordPress sites daily — this type of exposure is typically found within days of going live
  • Government vendor due diligence by primes (BAE, Lockheed, SPAWAR) runs security checks — this finding could fail a vendor qualification
Fix: Deactivate / delete plugin if not needed (~30 min). Or: update + require authentication on all API endpoints. Verify: visit /wp-json/ in incognito — file-manager-advanced/v1 should disappear or return 401/403.
MEDIUM Stale GSA Contract Credential on About Us Page

About Us page currently displays: "SpitzLift is a proud Contract Holder, License # GS-30F-0011U" — this contract was held by HIROK Inc. and expired/cancelled approximately March 2011, ~15 years ago. A contracting officer who verifies this credential will find it cancelled. Fix: 5 minutes in WordPress editor.

09 🛡️ Objection Handlers
"We're a small manufacturer — we don't need enterprise automation."
The automation isn't enterprise software — it's connecting the tools you already have. Gravity Forms → CRM → notification. Your team size is why this matters more, not less. You can't afford a 3-person follow-up team. The system does the follow-up.
"The website works fine. We get leads."
The website has an open security exposure right now that automated scanners are already looking for. That's not a lead generation problem — that's a liability. We can fix that in 30 minutes before we talk about anything else.
"I don't have budget for this right now."
Phase 1 is a flat-rate discovery engagement — scoped and capped. You're not buying a system, you're buying a plan. The plan tells you exactly what it costs to build and what the ROI is before you commit to anything.
"We already tried something like this and it didn't work."
Tell me what happened. Most implementations fail because they build around a new tool instead of an existing workflow. I build around what you already do — I don't throw away what works.
"Our IT guy handles the website."
This isn't an IT project — it's a business operations project. Your IT contact is my resource, not my client. I work with your IT person; the scope and ROI decisions come from you.
